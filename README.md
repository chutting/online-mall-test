# online-mall-test

## Project setup

> ```yarn install```

## Compiles and hot-reloads for development

> ```yarn dev```

## Compiles and minifies for production

>```yarn build```

## test

>```yarn test```

## 命名规则

- 文件夹名：中划线，首字母小写：`not-found`
- 组件名称：大驼峰：`NotFound`、`<NotFound>`
- 文件名： 小驼峰： `serviceStatus.ts`
